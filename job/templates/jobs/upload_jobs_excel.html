Excel File Format Instructions

Your Excel file should have these columns:

Required Columns:
  • description - Job description text
  • image - Full path to image file on server (e.g., "/home/user/images/job1.jpg")
Optional Columns:
  • category - Category name (will be created if doesn't exist)
  • video - Full path to video file on server
  • website_url - Website URL
  • whatsapp_number - WhatsApp number
  • facebook_url - Facebook URL
  • zoom_url - Zoom URL
  • microsoftTeam_url - Microsoft Teams URL
  • location - Job location
  • twitter_url - Twitter URL
  • playstore_url - Play Store URL
  • linkedin_url - LinkedIn URL
  • instagram_url - Instagram URL
  • pinterest_url - Pinterest URL
  • youtube_url - YouTube URL
File Path Note: The image and video paths should be absolute paths to files that exist on your server. The system will read these files and upload them to your media storage.
